Authorization

        

            This is authorization demo. You'll find two possible requests here:
            
    
                
  • Prompt for access - library will return you the link to
                    acquire access verifier so you can get your authorization token in the next request 
    
                    (also returns request information)
    • 
                
  • Request AccessToken - provide it with verifier (along with requestToken/secret so
                    the library knows it's the same request) and it will return you your authorization token
    • 
            

        
You can see how they are implemented in UserApiController.java#auth

        
Once you finish all your requests and get the authorization token,
            save it so you can execute user api requests in this demo
